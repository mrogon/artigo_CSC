# artigo_CSC
Este repostitório disponibiliza os dados de entrada, códigos e resultados do artigo "Impactos da pandemia na expectativa de vida da população no Brasil e no semiárido brasileiro entre 2010 e 2022"
Para replicabilidade dos resultados basta rodar o ódigo Decomposicao_Arriaga.R (Atenção para os comentários ao longo do código).
Para replicabilidade das Figuras, rodar o código Figuras_artigo.R
